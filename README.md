
# 🚆 Train Management System (Java)

## 📌 Overview
This project implements multiple use cases (UC1–UC20) to manage train operations using Java concepts like Collections, Streams, Searching, Sorting, and Exception Handling.

---

## ⚙️ Tech Used
- Java
- VS Code
- Git & GitHub

---

## 📂 Project Structure
Each use case is implemented in a **separate class file**:

UC1.java → UC20.java

---

## 🚀 Features (Use Cases)

- UC1: Initialize Train  
- UC2: Add Bogies (ArrayList)  
- UC3: Unique IDs (HashSet)  
- UC4: Sorted IDs (TreeSet)  
- UC5: Maintain Order (LinkedHashSet)  
- UC6: Bogie-Capacity Mapping (HashMap)  
- UC7: Sorting (Comparator)  
- UC8: Filtering (Streams)  
- UC9: Grouping Data  
- UC10: Total Seat Count (Reduce)  
- UC11: Input Validation  
- UC12: Safety Check  
- UC13: Performance Comparison  
- UC14: Invalid Data Handling  
- UC15: Exception Handling  
- UC16: Sorting (Collections)  
- UC17: Array Sorting  
- UC18: Linear Search  
- UC19: Binary Search  
- UC20: Exception Handling in Search
